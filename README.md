The **TCC (The Career Company) Campus Placement CRM** is a high-performance web application designed for campus placement teams (Placement Heads and Placecom Members). It streamlines the outreach process, manages corporate relationships, and tracks the placement funnel from initial contact to final offer.

### ðŸ§© Core Modules
- **Outreach CRM (`/dashboard`)**: Pipeline visualization, account management, and communication tracking (calls, emails, meetings).
- **Student Portal (`/student`)**: Student-facing interface for profiles and placement tracking.
- **Saarthi (`/saarthi`)**: AI-driven mock interviews, resume analysis, and career coaching tools.
- **Placement Officer (`/placement-officer`)**: Administrative tools for overall placement lifecycle control.

## ðŸ›  Getting Started

### Prerequisites
- **Node.js**: Version 18.x or above
- **Package Manager**: `npm` (native choice), `yarn`, or `pnpm`

### Installation
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Tccbymv/tcc-campus-placement-crm/
   cd v0-founder-guide-main
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

### Running Locally
To start the development server with Hot Module Replacement (HMR):
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.
